# КРАТКАЯ ПАМЯТКА: ПУТЬ КЛИЕНТА И AI-АВАТАРЫ

## ⚡ ПУТЬ КЛИЕНТА В ОДНУ СТРОКУ

```
ОХВАТЫ → ВОВЛЕЧЕНИЕ → КОММУНИКАЦИЯ → ЗАЯВКА → ПОКУПКА → ПРОДУКТ
  (1)        (2)            (3)           (4)       (5)       (6)
```

**Главное правило:** Нельзя сделать шаг 5 без 1-2-3-4!

---

## 🎯 ПОЗИЦИОНИРОВАНИЕ (3 вопроса)

```
1. КТО ВЫ?       → Ваша ниша
2. ДЛЯ КОГО?     → Ваша ЦА
3. ЧТО РЕШАЕТЕ?  → Проблема в 1-2 словах
```

### Формула шапки профиля:

```
[НИША] + [ДЛЯ КОГО] + [РЕЗУЛЬТАТ]

✅ Пример:
"Нейросети для бизнеса | 
Было 0 подписчиков → 295К за 2 года"
```

## 📐 СТРУКТУРА КОНТЕНТА

```
1. HOOK (3 сек)     → Захватить внимание
2. БОЛЬ (5-10 сек)  → Усилить проблему
3. РЕШЕНИЕ (30 сек) → Дать ценность
4. CTA (5 сек)      → Призыв к действию
```

### Формулы HOOK:

```
"Ты никогда не [РЕЗУЛЬТАТ], если не [ДЕЙСТВИЕ]"
"[ЦИФРА] признаков что ты [ПРОБЛЕМА]"
"Стоп! Если ты [БОЛЬ], смотри это"
```

### Формулы CTA:

```
"Пиши [СЛОВО], получишь [ПОЛЬЗА]"
"Сохрани + напиши [СЛОВО]"
"Подпишись, чтобы не потерять"
```

## 🤖 AI-АВАТАРЫ

### Два способа создания:

| Метод | Время | Качество | Сложность |
|-------|-------|----------|-----------|
| **Из картинки** | 2 мин | Хорошо | ⭐ Легко |
| **Из видео** | 5 мин | Отлично | ⭐⭐ Средне |

### 📸 Способ 1: Из картинки

```
1. Сгенерируйте фото (Magnific/Midjourney)
2. heygen.com → Upload look
3. Загрузите фото
4. Ждите 2 мин
5. Готово!
```

### 🎥 Способ 2: Из видео

```
1. Запишите себя 30-60 сек
   (говорите что угодно, смотрите в камеру)
2. heygen.com → Record now
3. Загрузите видео
4. Create look
5. Ждите 3 мин
6. Готово!
```

### 🎬 Создание видео:

```
1. Use in video
2. Вставьте текст (Hook-Боль-Решение-CTA)
3. Выберите голос
4. Generate
5. Скачайте
```
