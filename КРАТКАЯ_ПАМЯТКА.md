# КРАТКАЯ ПАМЯТКА: КОНТЕНТ И ПРОДАЖИ

## ⚡ БЫСТРЫЙ СТАРТ

### Формула денег в одну строку:
```
ОХВАТЫ (контент) → ВОВЛЕЧЕНИЕ (лид-магнит) → КОММУНИКАЦИЯ (директ) → ЗАЯВКА → ПРОДАЖА
```

---

## 🎯 КОНТЕНТ: ЧТО ПОСТИТЬ

### Соотношение типов контента:

```
📊 40-50% ШИРОКИЙ (тренды, новости ниши)
   └─ Цель: Максимум охватов

📚 40-50% ЭКСПЕРТНЫЙ (польза, обучение)
   └─ Цель: Экспертность + лояльность

💰 5-10% ПРОДАЮЩИЙ (оферы, призывы)
   └─ Цель: Заявки и продажи
```

### Структура КАЖДОГО поста:
1. **ХУК** (первые 3 сек) - привлечь внимание
2. **ПРОБЛЕМА** - описать боль
3. **РЕШЕНИЕ** - дать методологию  
4. **CTA** - сказать что делать

---

## 💬 ПРИЗЫВЫ К ДЕЙСТВИЮ (обязательно!)

### Где размещать:
✅ В самом видео (последний кадр)
✅ В описании поста
✅ В закреплённом комментарии

### Примеры:
```
"Напиши ХОЧУ в комментариях"
"Пиши слово УЧИСЬ, отправлю урок"
"Сохрани пост + напиши ГИД"
```

## 📊 ЛЕСТНИЦА ХАНТА (как вести клиента)

```
90% АУДИТОРИИ        →  Контент: "У тебя есть проблема!"
  ↓ Проблема осознана
  
                    →  Контент: "Вот решение"
  ↓ Решение найдено
  
                    →  Контент: "Почему я лучше других"
  ↓ Сравнение
  
3% АУДИТОРИИ        →  Контент: "Купи сейчас"
  ↓ ПОКУПКА 💰
```

**Главное:** Делайте контент для 90%, а не для 3%!

## 💡 ВАЖНЫЕ ЦИФРЫ

### Воронка продаж:
```
1,000,000 просмотров
    ↓ 1%
10,000 вовлеклись
    ↓ 5%
500 в коммуникации
    ↓ 10%
50 заявок
    ↓ 40%
20 купили

ИТОГО: 0.002% конверсия = НОРМАЛЬНО
```

### Бюджет кадров для видео:
- ≤30 сек: ~30 кадров (плотно)
- 1-3 мин: ~60 кадров (комфортно)
- 3-10 мин: ~80 кадров (sparse)
- \>10 мин: Фокус на отрезок
