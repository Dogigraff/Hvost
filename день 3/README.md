## 🎯 Ключевые концепции

### Формула денег:
```
ОХВАТЫ → ВОВЛЕЧЕНИЕ → КОММУНИКАЦИЯ → ЗАЯВКА → ПРОДАЖА
```

### Правило 90/10:
- **90%** контента для тех, кто НЕ знает о проблеме
- **10%** контента для тех, кто готов купить

### Типы контента:
- **40-50%** - Широкий (тренды, новости)
- **40-50%** - Экспертный (польза, обучение)
- **5-10%** - Продающий (оферы)
